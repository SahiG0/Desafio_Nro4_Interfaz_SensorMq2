Resumen proyecto: Nuestro proyecto constara en realizar una interfaz amigable con el usuario capaz de mostarar los datos recibidos por el sensor MQ2
con su respectivo front end y back end.

Objetivos:
    1.Investigar UX/UI.
    2.Desarrollar una interfaz gráfica para visualizar los datos transmitidos por el sensor MQ2 mediante Bluetooth.
    3.Comprender la recepción de datos desde dispositivos Arduino en una PC.
    4.Aplicar conocimientos de programación en Python o en tecnologías web (HTML, CSS y JavaScript) para crear interfaces de usuario.
